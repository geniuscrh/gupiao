## 安装TensorFlow

```python
#创建环境
#python3.5环境最稳定(网传)
conda create --name tensorflow python=3.5
#激活环境
activate tensorflow
#升级pip,百度搜索"pip 镜像"
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pip -U
#下载安装tensorflow
pip install tensorflow
```
